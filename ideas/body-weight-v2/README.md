# body-weight-v2

Body Weight の version 2.0 として、以下を1画面で記録・確認できるWebアプリです。

- 体重 (kg)
- 摂取カロリー (kcal)
- 活動消費カロリー (kcal)

入力データはブラウザの `localStorage` に保存されます。

## 起動方法

サーバー不要です。`index.html` をブラウザで開くだけで動きます。

例（このリポジトリ直下で実行）:

```bash
python3 -m http.server 8000
```

その後、`http://localhost:8000/ideas/body-weight-v2/` を開いてください。

## 確認方法

1. 日付・体重・摂取カロリー・活動消費カロリーを入力して「保存 / 更新」。
2. 履歴テーブルに追加されることを確認。
3. グラフ（3系列）が更新されることを確認。
4. 同じ日付で再保存すると上書きされることを確認。

## 次に足すとよい機能

- 体重だけ右軸に分ける（kcalと単位が違うため見やすくなる）。
- 週次/月次ビュー切替。
- 目標体重・目標収支ラインの表示。
