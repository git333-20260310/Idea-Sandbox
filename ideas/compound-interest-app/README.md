# 複利計算アプリ

初期資金・毎月の積立額・年利・運用年数を入力して、将来の資産額を試算するシンプルなWebアプリです。

## 起動方法

このリポジトリのルート（`Idea-Sandbox`）で次を実行します。

```bash
python3 -m http.server 8000
```

ブラウザで以下を開きます。

- `http://localhost:8000/ideas/compound-interest-app/`

## 確認方法

1. 初期資金・積立額・年利・運用期間を入力して「計算する」を押す。
2. 「計算結果」に将来評価額、元本+積立総額、運用益が表示されることを確認する。
3. 下部の「年ごとの推移」テーブルが更新されることを確認する。
4. ページ再読み込み後も前回の入力値が保持される（localStorage）ことを確認する。
