# first-test-page

iPhoneから確認するための最小テストページです。

## GitHub Pagesで開く

このフォルダをGitHubにpushしたあと、GitHub Pagesを有効化して `ideas/first-test-page/index.html` を公開してください。

## ローカル確認

`ideas/first-test-page/index.html` をブラウザで直接開くだけで確認できます。

## 確認ポイント

- 画面に `Idea-Sandbox test` が表示される
- 今日の日付（YYYY-MM-DD）が表示される
- ボタンを押すたびに表示テキストが切り替わる

## 次に足すとよい機能

- ボタン押下回数の表示
- ダークモード切り替え
