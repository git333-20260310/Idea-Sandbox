# tap-counter

iPhoneで使いやすい、シンプルなタップカウンターです。  
「タップ +1」を押すたびに数字が増え、値はブラウザに保存されます。

## 起動方法

GitHub Pagesで `ideas/tap-counter/` を公開すると、そのまま動きます。

ローカル確認の場合:

```bash
cd ideas/tap-counter
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いて確認してください。

## 確認方法

1. 「タップ +1」を押して数字が増えることを確認
2. ページを再読み込みして値が保持されることを確認
3. 「リセット」で `0` に戻ることを確認
